# Secure-RAG-Pipeline-for-Information-Security-Tasks

## 📌 Описание
Данный репозиторий содержит реализацию защищенного пайплайна **Retrieval-Augmented Generation (RAG)** для задач информационной безопасности. Проект разработан в рамках курсовой работы по дисциплине «Искусственный интеллект в защите информации» (Вариант 64, Уровень С).

Основная цель системы — предоставление точных ответов на вопросы по нормативным документам (приказы ФСТЭК, ГОСТы, ФЗ) с внедрением многоуровневых механизмов защиты от атак на языковые модели (Prompt Injection, Data Leakage).

## 🛠 Технологический стек

## 🚀 Ключевые особенности
### 1. Защита входных данных (Input Guard)
- Обнаружение Prompt Injection (jailbreak-шаблоны, игнорирование инструкций)
- Фильтрация запрещенного контента (запросы на создание вредоносного ПО)
- Блокировка попыток извлечения данных (Data Extraction)

### 2. Ограничение контекста (Context Limiter)
- Дедупликация источников
- Пороговая фильтрация релевантности (min_score=0.3)
- Ограничение объема передаваемого контекста

### 3. Фильтрация вывода (Output Guard)
- Поиск и маскировка чувствительных данных (номера приказов, ФИО, даты, IP-адреса, пароли)

### 4. Сравнительный анализ LLM
- Сравнение производительности Qwen2.5-3B vs Llama3.2-3B
- Оценка точности, полноты и робастности к атакам

## 📊 Результаты экспериментов

### Качество поиска (Information Retrieval)
| Метрика | Значение |
|---------|----------|
| MRR (Mean Reciprocal Rank) | 0.678 |
| NDCG@3 | 0.687 |
| Precision@3 | 0.489 |
| Recall@3 | 0.810 |

### Сравнение моделей (Qwen vs Llama)
| Модель | P@3 | R@3 | Защита от инъекций | Предотвращение утечек |
|--------|-----|-----|-------------------|----------------------|
| Qwen2.5-3B | 0.87 | 0.82 | 96% | 100% |
| Llama3.2-3B | 0.85 | 0.80 | 94% | 100% |

### Результаты безопасности
| Тип атаки | Заблокировано |
|-----------|---------------|
| Prompt Injection (прямые инъекции) | 70% |
| Jailbreak (DAN и аналоги) | 65% |
| Data Leakage (извлечение документов) | 50% |
| Системный промпт extraction | 100% |
